## 安装

1. 通过`pip`或`nb`安装，版本请指定`0.2.4`；

2. 塔罗牌图片资源默认位于`./resource`下，设置`env`下`TAROT_PATH`更改资源路径，`CHAIN_REPLY`设置启用转发模式；

3. 占卜一下你的今日运势！🥳

## 功能

1. 塔罗牌！占卜今日运势；

2. 支持群聊/私聊占卜；

3. 支持单张/全套塔罗牌占卜；

4. **新增** 插件帮助文案；

## 命令

1. 占卜：占卜4张塔罗牌；

2. 塔罗牌：得到单张塔罗牌回应；
